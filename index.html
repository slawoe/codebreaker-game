Description

The game is quite simple.
You have to solve the hidden color code behind the gray fields.
You have to fill the empty (white) solution fields with colors. Click in the colorpicker on the desired color and then on the desired empty field.
The colors can be placed at any position and can be used multiple times. You can change the color at any time.
Once you are done, you can use the Solve button to check if your answer is correct.
You will get as a little help how many fields you got right, but not which ones.
How many tries will you need?
PS: There are no empty (white) fields!
